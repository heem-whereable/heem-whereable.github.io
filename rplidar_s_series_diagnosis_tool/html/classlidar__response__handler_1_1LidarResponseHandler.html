This class handles the responses from the LiDAR device.
This class is responsible for parsing the responses from the LiDAR device and updating the monitor accordingly.

def lidar_response_handler.LidarResponseHandler.parse_express_scan_response (   self,
  response

Parses the express scan response.
This function parses the express scan response and updates the monitor accordingly.
The express scan response is a continuous stream of data.
The response is parsed by extracting the packets from the stream and updating the monitor with the Cartesian coordinates.

Response packet format: 

|---------------------------------------------  Express Scan  ---------------------------------------------|
|                                                                                                          |
| Sync 1 | checksum 1 | Sync 2 | checksum 2 | Start Angle 1 | isStarting | Start Angle 2 | Distance Cabins |
|  4 bit |   4 bit    | 4 bit  |   4 bit    |     8 bit     |   1 bit    |    7 bit      |     80 bit      |
|       1 byte        |       1 byte        |     1 byte    |         1 byte             |     40 byte     |
|---------------------------------------------    Example    ----------------------------------------------|
|  0xA0  |    0x0F    |  0x50  |    0x0D    |      0xF6     |     0x0    |     0x39      |      ...        |
|              Checksum = 0xDF              |         Angle = (0x39 << 7) + 0xF6 = 0x0EF6           |
|----------------------------------------------------------------------------------------------------------|
|                                            Header + Cabins                                               |
|                                           4 + 80 = 84 bytes                                              |
