\doxysubsection{Detailed Description}
\begin{DoxyVerb}This class represents the main window of the application. It contains the control panel, command panel, LiDAR monitor, and status monitor.
each of these components is a separate tab in the main window. The main window also contains the menu bar.
\end{DoxyVerb}
