\doxysubsection{Detailed Description}
\begin{DoxyVerb}This is the main file that runs the application.
It creates a QApplication instance and sets the asyncio event loop to the QEventLoop.
It then creates the main window and shows it. Finally, it starts the event loop.
This is because PyQt6 is not fully compatible with asyncio, so we need to use the qasync library to run the event loop.\end{DoxyVerb}
